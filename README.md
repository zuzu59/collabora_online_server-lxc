# collabora_online_server-lxc
Installation de Collabora Online server dans un container LXC sous Proxmox

zf250808.1506


## Utilisation
Pour démarrer simplement faire:
```
./start.sh
```

Pour arrêter simplement faire:
```
./stop.sh
```

Pour updater simplement faire:
```
./update.sh
```
